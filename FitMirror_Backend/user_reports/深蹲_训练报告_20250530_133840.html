关键错误诊断
在您的深蹲训练中检测到膝盖内扣的问题。这可能是由于髋部稳定性不足、腿部肌肉力量不平衡或动作模式不正确导致的。膝盖内扣会增加膝关节的压力，可能导致受伤风险上升。建议您调整姿势并加强相关肌肉群的力量。
⚠️ 膝盖内扣: 检测到 1 次
• 重心前移: 抬高脚跟或穿戴举重鞋（持续）
• 腰椎超伸: 练习腹式呼吸+核心绷紧×1次

【退阶方案】
📚 确保脚尖略微向外打开（约30度），以帮助膝盖与脚趾方向一致。
📚 练习臀桥或侧卧抬腿，增强臀部和髋外展肌的力量。
📚 尝试无重量的空气深蹲，专注于保持膝盖对准脚趾的动作轨迹。
📚 在镜子前进行深蹲，观察膝盖位置并及时调整。
【进阶方案】
🚀 加入单腿深蹲（保加利亚分裂蹲）来提高单侧稳定性和力量。
🚀 使用弹力带环绕膝盖上方，强化髋部外展肌并纠正膝盖内扣的习惯。
🚀 尝试暂停深蹲，在最低点停留2-3秒，增强控制力和核心稳定性。
🚀 定期拉伸髂胫束和内收肌群，改善柔韧性并减少膝盖内扣的可能性。
